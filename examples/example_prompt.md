# 示例提示词

请你调研日本声优【声优名】在【年份】的大致年收入，单位日元。

要求：

1. 默认口径为“事务所抽成后、个人所得税和社保前的个人税前收入”。
2. 从以下维度拆解：
   - 人气与商业化画像
   - 长期饭票
   - 短期业务
   - 动画 / 外画配音
   - 游戏录音
   - 广播 / 网络节目
   - 会员订阅
   - live / concert / festival
   - 演出分成 / 配信 / 物贩 bonus
   - 角色歌 / 音乐
   - 联名商品 / 周边 / voice 商品
   - CM / 旁白 / 广告
   - IP 分成
   - 事务所分成和抽成
3. 所有公开事实需要给出来源。
4. 所有模型假设必须单独列出，不要写成事实。
5. 给出保守、基准、乐观三种情景。
6. 给出敏感性分析，说明最影响最终收入的参数。
7. 最后给出置信度和主要不确定性。

输出格式：

- 结论
- 商业化画像
- 公开活动与收入抓手
- 收入模型表
- 关键假设
- 敏感性分析
- 置信度
